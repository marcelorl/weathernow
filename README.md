# weathernow

## Overview

Application to show the current weather of:
 - Uribici, BR;
 - Nuuk, GL;
 - Nairobi, KE
 
## Tools

 - React
 - Redux
 - styled-components
 - jest
 - Atomic Design
 - Openweather api
 
## Installation

```
$ npm i
```
 
## Usage
 
```
$ npm start
```
