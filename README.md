# weathernow

Application to show the current weather of:
 - Uribici, BR;
 - Nuuk, GL;
 - Nairobi, KE
 
### Tools

 - React
 - Redux
 - styled-components
 - jest
 - Atomic Design
 - Openweather api
 
### Installation

```
$ cp .env.local.example .env.local // paste your openweather id here
$ npm i
```
 
### Usage
 
```
$ npm start
```

### Test

```
$ npm test
```
